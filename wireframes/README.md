NovaMart — Wireframes
======================

About this folder
-----------------
Wireframes were produced as high-fidelity reference screens using
Cursor AI, driven by detailed UX prompts built directly from the
persona research (Day 2) and journey maps (Day 3).

Each prompt specified:
  - Exact screen layout zones (header, content, nav)
  - Content hierarchy and element order
  - Specific persona pain points to address
  - Design system colours, font, and frame size

The Figma screens in the prototype were rebuilt from these references
using the NovaMart Design System (see /design-system/).
